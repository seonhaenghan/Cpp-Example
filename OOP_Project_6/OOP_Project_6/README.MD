# OOP 프로젝트 01 단계

## 프로그램 설명
>> 구현하고자 하는 것은 '은행 계좌 관리 프로그램'
	- 기능 1. 계좌개설
	- 기능 2. 입    금
	- 기능 3. 출    금
	- 기능 4. 전체고색 잔액조회 

프로그램을 간결히 하기 위한 몇 가지 가정 

	- 통장의 계좌번호는 중복되지 아니한다.(중복검사를 하지 않음)
	- 입금 및 출금액은 무조건 0보다 크다(입금 및 출금액의 오류검사를 하지 않음)
	- 고객의 계좌정보는 계좌번호, 고객이름, 고객의 잔액, 이렇게 세 가지만 저장 및 관리
	- 둘 이상의 고객 정보 저장을 위해서 배열을 사용
	- 계좌번호는 정수의 형태
